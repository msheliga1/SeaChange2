Architecture Overview

This platform is a web application. Below are some bullet points on its most important features:

  1. The essential technology stack used is Javascript (JQuery & Bootstrap) / Spring / Hibernate / MySQL.
  2. It is assumed that each organization running the system will have their own installation of it running on a cloud server.
  3. Plugins are used add functionality to online roleplays.
